# Next Level Scheduler

Scheduler web app for Next Level staff:

- Table view + calendar view of schedules
- A **coverage / swap** workflow so employees can take time off when someone can cover
- Provider scheduling constraints (each provider has a medical assistant assigned on specific days)

This repo uses a Cursor-first “Website Development Pipeline”. The current state and next actions live in `PROJECT_WEB_DEV_STATE.md`.

## How to work in this repo

1. In a new Cursor chat, say: “Follow `website-development-orchestrator` and read `PROJECT_WEB_DEV_STATE.md` first.”
2. Continue from **Active phase** and the single **CURRENT** item.
